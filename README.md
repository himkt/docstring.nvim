# Install

### vim-plug

```viml
Plug 'himkt/docstring.nvim', { 'do': ':UpdateRemotePlugins' }
```

### dein.vim

You may have to run `:UpdateRemotePlugins` manually.

```viml
call dein#add('Shougo/deoplete.nvim')
```
